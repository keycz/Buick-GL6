电路/系统说明
加热型氧气传感器(HO2S) 1或2用于燃油控制和催化剂监测。每个加热型氧传感器将环境空气的氧含量与废气流中的氧含量进行比较。起动发动机，控制模块在开环模式下工作，当其计算空燃比时，忽略加热型氧气传感器的信号电压。控制模块给加热型氧气传感器提供参考电压，或约450毫伏的偏压。发动机运行时，每个加热型氧气传感器开始加热，并产生在0 - 1,000毫伏范围内的电压。该电压将在偏压值上下波动。控制模块检测到经过充分加热的氧气传感器1或2的电压波动后，则进入闭环模式。控制模块通过加热型氧气传感器的电压来确定空燃比。加热型氧气传感器的电压值高于偏压指向1000毫伏时，说明混合气为富油混合气。加热型氧气传感器的电压值低于偏压指向0毫伏时，说明混合气为贫油混合气。
各个加热型氧传感器内的加热元件对传感器加热，使其迅速升高至工作温度。这就使得系统能更早地进入“闭环”模式，并使模块更快地计算空燃比。
运行DTC的条件

    P2096 00

  • •  DTC P0030、P0031、P0032、P0130、P0131、P0132、P0133、P0135、P0137、P0138、P0140、P2270或P2271未设置。
  • •  发动机转速在1,520 - 3,520转/分之间。
  • •  点火电压在10-18伏之间。
  • •  发动机负载在16 - 21%之间。
  • •  闭环燃油控制激活并持续5秒以上。
  • •  蒸发排放(EVAP) 系统不吹洗。
  • •  计算的排气温度高于250°C (482°F)
  • •  一旦上述条件满足并持续100秒钟，故障诊断 码将持续运行。

    P2097 00

  • •  DTC P0030、P0031、P0032、P0130、P0131、P0132、P0133、P0135、P0137、P0138、P0140、P2270或P2271未设置。
  • •  发动机转速在1,520 - 3,520转/分之间。
  • •  点火电压在10-18伏之间。
  • •  发动机负载在17 - 20%之间。
  • •  蒸发排放系统不吹洗。
  • •  闭环燃油控制激活并持续5秒以上。
  • •  一旦上述条件满足并持续100秒钟，DTC将持续运行。
设置DTC的条件

P2096 00

当加热型氧气传感器电压试着按相同的方向进一步调节时，长期燃油调节到达下限。

P2097 00

当加热型氧气传感器电压试着按相同的方向进一步调节时，长期燃油调节达到上限。
设置DTC时采取的操作
DTC P2096、P2097是B类DTC。
清除DTC的条件
DTC P2096、P2097是B类DTC。

电路/系统检验
  • 1.将点火开关置于“ON（打开）”位置。
  • 2.确认故障诊断仪上认没有设置其他加热型氧传感器或燃油调节DTC。
  • 如果显示故障码
    参见“故障诊断码（DTC）列表－车辆”以作进一步诊断。
  • 如果未显示故障
  • 注意:为了运行该诊断，B52B加热型氧气传感器2必须处于闭环模式。要获得闭环模式，必须具有道路负荷条件。
    3.使发动机达到工作温度。
  • 4.在“运行DTC的条件”下操作车辆，或在从“冻结故障状态/故障记录”数据中查到的条件下操作车辆。确认未设置DTC。
  • 如果设置了DTC
    参见“电路/系统测试”。
  • 如果未设置DTC
  • 5.全部正常。
电路/系统测试
注意:所有模块必须断电，否则可能导致误诊。
  • 1.确认不存在以下状况：
    • •  检查B52A加热型氧气传感器1或B52B加热型氧气传感器2是否紧固
    • •  B52A加热型氧气传感器1或B52B加热型氧气传感器2与发动机控制模块之间的线束损坏
    • •  B52A加热型氧气传感器1或B52B加热型氧气传感器2线束连接器进水
    • •  燃油污染，参见酒精/污染物进入燃油的诊断。
    • •  蒸发排放(EVAP) 炭罐的燃油饱和
    • •  B52A加热型氧气传感器1或B52B加热型氧 气传感器2附近的排气系统泄漏
    • •  发动机真空泄漏
    • – 如果发现上述任何情形，必要时进行 维修。
  • 2.将点火开关置于OFF（关闭）位置，断开B52B加热型氧传感器2线束连接器。
  • 3.将点火开关置于“ON（打开）”位置，确认故障诊断仪的“HO2S Sensor 2（加热型氧气传感器2）”电压参数在350 - 550毫伏之间。
  • 如果不在规定范围
    • 3.1 将点火开关置于“OFF（关闭）”位置，断开K20发动机控制模块处的线束连接器。
    • 3.2 测试每一相关部件所对应的5伏参考电压电路和搭铁之间的电阻是否为无穷大。
    • 如果电阻不为无穷大，则修理电路上的对搭铁短路故障。
    • 如果电阻为无穷大，则更换K20“发动机控制模块”。
  • 如果在规定范围内
  • 4.连接B52B加热型氧传感器2处的线束连接器。
  • 5.发动机怠速运行时，将发动机转速提高至大于1,200转/分并持续30秒。确认故障诊断仪的“HO2S Sensor 2（加热型氧气传感器2）”电压参数高于60毫伏。
  • 如果低于规定范围
    更换B52B加热型氧传感器2
  • 如果在规定范围内
  • 6.发动机怠速运转时，移动加热型氧气传感器线束连接器和K20发动机控制模块之间的B52A加热型氧气传感器1或B52B加热型氧气传感器2线束。观察故障诊断仪的“HO2S Sensor 1（加热型氧气传感器1）”和“HO2S Sensor 2（加热型氧气传感器2）”电压参数。确认参数未出现突然变化。
  • 如果参数突然变化
    更换线束/连接器。
  • 如果测试正常
  • 7.更换B52B加热型氧传感器2。
