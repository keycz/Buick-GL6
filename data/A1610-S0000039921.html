电路/系统说明
每个轮速传感器通过高电平信号电路从电子制动控制模块处接收点火电压。轮速由主动式轮速传感器和编码器环检测。磁性编码器环由多个磁南级和磁北极对围绕着外周组成。当编码器环转动并通过轮速传感器时，在轮速传感器内产生信号波。传感器将信号转换为数字方波；方波值在7毫安（低）和14毫安（高）直流电之间转变。当车轮旋转时，该模块通过此方波信号的频率来计算轮速。

设置DTC时采取的操作
  • •  防抱死制动系统指示灯点亮。
  • •  牵引力/稳定性控制指示灯点亮。
  • •  可能显示驾驶员信息中心的信息。
  • •  在点火循环期间，电子制动控制模块停用防抱死制动系统、牵引力控制系统和电子稳定性程序。
清除DTC的条件
  • •  设置该DTC的条件不再存在时。
  • •  经过40个连续无故障点火循环后，历史DTC将被清除。
诊断帮助
  • •  切勿使用磁铁清理编码器环。
  • •  检查车轮编码器环是否锈蚀。
  • •  检查轮速传感器和部件是否为正确的零件。
  • •  如果2个或多个轮速传感器不工作，则需单独诊断每个轮速传感器。
  • •  如果设置了2F、37或64症状编码，则参见“诊断故障信息”表，以了解可能的机械故障或情况。
  • •  如果客户的故障描述反映防抱死制动系统指示灯只在潮湿环境下（如雨天、雪天、洗车时）点亮，则检查所有轮速传感器接线是否有进水迹象。

电路/系统检验
  • 1.以高于40公里/小时（25英里/小时）的速度直线驾驶车辆。
  • 2.确认故障诊断仪所有“Wheel Speed Sensor（轮速传感器）”参数彼此相差应在1.6公里/小时（1英里/小时）内。
  • 如果任何参数彼此相差不在1.6公里/小时（1英里/小时）内。
    参见“电路/系统测试”。
  • 如果每个参数彼此相差在1.6公里/小时（1英里/小时）内。
  • 3.全部正常。
电路/系统测试
  • 1.将点火开关置于“OFF（关闭）”位置，并关闭所有车辆系统，断开相应B5“轮速传感器”的线束连接器及K17“电子制动控制模块”的线束连接器，然后将点火开关置于“ON（打开）”位置。
  • 2.测试下列K17电子制动控制模块线束连接器端子和搭铁之间的电压是否低于1伏：
    • •  B5LF 轮速传感器 - 左前 - 端子24和7
    • •  B5LR 轮速传感器 - 左后 - 端子39和23
    • •  B5RF 轮速传感器 - 右前 - 端子21和26
    • •  B5RR 轮速传感器 - 右后 - 端子22和37
  • 如果等于或大于1伏
    修理电路上的对电压短路。
  • 如果低于1伏
  • 3.点火开关置于OFF（关闭）位置。
  • 4.测试下列K17“电子制动控制模块”线束连接器端子和搭铁之间的电阻是否为无穷大：
    • •  B5LF 轮速传感器 - 左前 - 端子24和7
    • •  B5LR 轮速传感器 - 左后 - 端子39和23
    • •  B5RF 轮速传感器 - 右前 - 端子21和26
    • •  B5RR 轮速传感器 - 右后 - 端子22和37
  • 如果电阻不为无穷大
    修理电路上的对搭铁短路。
  • 如果电阻为无穷大
  • 5.测试B5“轮速传感器”线束连接器端子1和下列K17“电子制动控制模块”线束连接器端子之间的电阻是否小于2欧：
    • •  B5LF 轮速传感器 - 左前 - 端子24
    • •  B5LR 轮速传感器 - 左后 - 端子39
    • •  B5RF 轮速传感器 - 右前 - 端子21
    • •  B5RR 轮速传感器 - 右后 - 端子22
  • 如果等于或大于2欧
    修理电路中的开路/电阻过大。
  • 如果小于2欧
  • 6.测试B5“轮速传感器”线束连接器端子2和下列K17“电子制动控制模块”线束连接器端子之间的电阻是否小于2欧：
    • •  B5LF 轮速传感器 - 左前 - 端子7
    • •  B5LR 轮速传感器 - 左后 - 端子23
    • •  B5RF 轮速传感器 - 右前 - 端子26
    • •  B5RR 轮速传感器 - 右后 - 端子37
  • 如果等于或大于2欧
    修理电路中的开路/电阻过大。
  • 如果小于2欧
  • 7.更换B5“轮速传感器”。
  • 8.在“运行DTC的条件”下操作车辆，确认未设置DTC。
  • 如果设置了DTC
    更换K17电子制动控制模块。
  • 如果未设置DTC
  • 9.全部正常。
