K73 远程通信接口控制模块：编程和设置

    注意:
  • •  切勿对控制模块进行编程，除非有维修程序或维修通讯的指导。如果控制模块未利用正确的校准软件进行正确配置，则控制模块无法正确控制所有车辆功能。
  • •  确保编程工具装备了最新软件并且与数据链路连接器牢固连接。如果编程过程中断，则可能会使编程失败或损坏控制模块。
  • •  在编程过程中，蓄电池电压稳定很重要。任何波动、峰值、过压或电压损失都将中断编程。需要时，安装EL-49642维修编程系统编程支持工具以维持系统电压。如果不适用，则连接一根电压充足的12 V跨接线或从交流电压电源上断开的助力器组件。切勿连接蓄电池充电器。
  • •  关闭或停用可能对车辆蓄电池造成负荷的系统，如车内灯、车外灯（包括日间行车灯）、HVAC、收音机等。
  • •  在编程程序期间，遵循SPS提示以修正点火开关位置。
  • •  完成编程后，清除DTC。清除动力总成 DTC 可能会将“Inspection/Maintenance (I/M)（检查/保养）”系统状态指示器设置为“NO（否）”。
诊断帮助
  • •  开始任何编程工作前，先确定您面对的是哪一代远程通信接口控制模块。实现这个目标的方法之一是使用故障诊断仪。远程通信接口控制模块代将在所选车辆配置部分下方列出，另外可从模块 ID 数据表中确定是那一代（9 或 10）。根据模块的代数，可能需要其他编程步骤。
  • •  至关重要的是，仅在订购维修更换通信接口模块的车辆内使用该模块。
  • •  如果无法执行以下程序，将会导致红色LED点亮、设置DTC且安吉星®服务受限或不完整。
  • •  需按下安吉星®按钮进入安吉星®呼叫中心来完成此程序。
  • •  要完全激活安吉星®模块，必须在带和不带有效安吉星®预约服务的所有车辆上完成设置和激活请求程序，以确保已安装至车辆的装置已经正确设置，并用正确的装置信息更新安吉星®。
  • •  如果车辆配备有常规选装件 VV4，则必须在编程和维修激活步骤后执行远程通信接口控制模块 WiFi 启用（仅第 10代 ）的编程步骤。如未按照正确的顺序完成此步骤，会导致Wifi被停用。
  • •  要初始化转弯功能，则车辆必须首先以大于10mph（16km/h）或更高车速在露天条件下行驶最少 5 mi (8 km)，然后在每次转向之前执行最少2次90°左转和2次90°右转至完全停止。
  • •  对于配备TTY功能的车辆，必须对安吉星®接口模块（OTIM）执行一个通电循环，并在完成通信接口模块的安装和设置程序后切换至TTY模式。

更换和编程控制模块，或对其重新编程
要对一个更换的或已有的控制模块进行编程，则执行以下程序：
  • 1.安装EL-49642维修编程系统编程支持工具。
  • 2.进入维修编程系统 (SPS)，并按屏幕上的说明进行操作。
      • 注意:
      • •  由于维修编程系统的更改，对于“Telematics Communication Interface Control Module Programming from the Selected Programming Process（从选定的编程过程对远程通信接口控制模块编程）”，即使发动机控制单元将更换，也必须选择“Reprogram ECU（重新编程ECU）”。
      • •  如果选择了“Replace and Program ECU（更换和编程 ECU）”，则远程通信接口控制模块不会出现在“Supported Controllers（支持的控制器）”屏幕上。必须返回并选择“Reprogram ECU（重新编程ECU）”。
  • 3.在“SPS Supported Controllers（维修编程系统支持的控制器）”屏幕上，选择“Telematics Communication Interface Control Module Programming and Service Activation（远程通信接口控制模块编程和服务激活）”，并按屏幕上的说明进行操作。
  • 4.在编程结束后，选择维修编程系统屏幕上的“Clear All DTCs（清除所有DTC）”功能。
  • 5.远程通信接口控制模块编程和维修激活后，如果车辆装备常规选装件 VV4，在支持的控制器屏幕选择“Telematics Communication Interface Control Module WiFi Enable（Gen 10 only）Programming（远程通信接口控制模块 WiFi 启用（仅第 10代 ）编程）”并按屏幕上的说明进行操作。
    注意:WiFi启用步骤必须在编程和维修激活步骤后执行，否则WiFi将不工作。如果编程和维修激活步骤在WiFi启用步骤后执行，则WiFi将被停用。
  • 6.按下安吉星®蓝色按钮，可呼叫安吉星®，请他们激活系统。如果客户拒绝安吉星®服务（条款和协议），但车辆配备常规选装件VV4，则请安吉星®顾问关闭WiFi演示信息（每次将点火开关置于ON（打开位置）事件后出现）。
编程恢复不成功
如果编程操作中断或失败，则执行以下步骤：
  • 1.将点火开关置于“ON（打开）”位置。确保控制模块、数据链路连接器和编程工具连接牢靠且维修编程系统软件是最新版本。
  • 2.确认控制模块能够重新编程。
  • 如果控制模块无法重新编程
    • 2.1 将点火开关置于“OFF（关闭）”位置 1 min，然后将点火开关置于“ON（打开）”位置。
    • 2.2 确认控制模块能够重新编程。
    • 如果控制模块无法重新编程，则更换控制模块。
    • 如果控制模块能够重新编程。
    • 2.3 一切正常。
  • 如果控制模块能够重新编程
  • 3.一切正常。
