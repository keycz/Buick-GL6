电路说明
故障指示灯(MIL) 位于仪表板组合仪表(IPC) 上。故障指示灯通知驾驶员，出现了排放系统故障并且发动机控制系统需要维修。发动机控制模块(ECM) 监测故障指示灯控制电路，是否有与故障指示灯指令状态不一致的故障。例如，如果在故障指示灯被指令熄灭时发动机控制模块检测到电压过低，或者故障指示灯被指令点亮时检测到电压过高，这表明存在故障。
运行DTC的条件
  • •  发动机转速高于80 转/ 分。
  • •  点火电压介于9-16 伏之间。
  • •  在点火循环中，发动机控制模块已指令故障指示灯点亮和熄灭至少一次。
  • •  满足上述条件时， DTC P0650 将持续运行。

电路/ 系统检验
  • 1.将点火开关置于“ON （打开）”位置。
  • 2.用故障诊断仪指令故障指示灯点亮和熄灭。
  • 3.确认故障指示灯按照指令点亮、熄灭。
  • 如果故障指示灯未按照指令点亮、熄灭
    参见“电路/系统测试”
  • 如果故障指示灯按照指令点亮、熄灭
  • 4.将点火开关置于“ON （打开）”位置。
  • 5.用故障诊断仪指令故障指示灯点亮和熄灭，并确认下列控制电路状态参数显示“OK （正常）” 或“Not Run （未运行）”。
    • •  MIL Control Circuit Low Voltage Test Status （故障指示灯控制电路电压过低测试状态）
    • •  MIL Control Circuit Open Test Status（故障指示灯控制电路开路测试状态）
    • •  MIL Control Circuit High Voltage Test Status （故障指示灯控制电路电压过高测试状态）
  • 如果显示故障
    参见“电路/系统测试”
  • 如果显示“OK （正常）”或“Not Run （未运行）”
  • 6.发动机运转。
  • 7.用故障诊断仪指令故障指示灯点亮和熄灭，并确 认下列控制电路状态参数显示“OK （正常）” 或“Not Run （未运行）”。
    • •  MIL Control Circuit Low Voltage Test Status （故障指示灯控制电路电压过低测试状态）
    • •  MIL Control Circuit Open Test Status（故障指示灯控制电路开路测试状态）
    • •  MIL Control Circuit High Voltage Test Status （故障指示灯控制电路电压过高测试状态）
  • 如果显示故障
    参见“电路/系统测试”
  • 如果显示“OK （正常）”或“Not Run （未运行）”
  • 8.在“运行DTC的条件”下操作车辆，或在从“冻结故障状态/ 故障记录”数据中查到的条件下操作车辆。
  • 如果设置了DTC
    参见“电路/系统测试”
  • 如果未设置DTC
  • 9.全部正常。
电路/ 系统测试
  • 1.将点火开关置于“OFF （关闭）”位置。
  • 2.断开K20 发动机控制模块处的X1 线束连接器。
  • 3.将点火开关置于“ON （打开）”位置。
  • 4.确认故障指示灯未点亮。
  • 如果故障指示灯点亮
    • 4.1 测试故障指示灯控制电路端子X1 D4和搭铁之间的电阻是否为无穷大。
    • 如果电阻不为无穷大，则修理电路上的对搭铁短路故障。
    • 如果电阻无穷大，更换P16 组合仪表。
  • 如果故障指示灯未点亮
  • 5.在故障指示灯控制电路端子X1 D4和搭铁之间连接一根带3 安培保险丝的跨接线。
  • 6.确认故障指示灯点亮。
  • 如果故障指示灯未点亮
    • 6.1 测试控制电路端子 X1 D4和搭铁之间的电压是否小于1 V。
    • 如果等于或大于1 V，则修理电路中的对电压短路故障。
    • 如果小于1 V
    • 6.2 将点火开关置于“OFF（关闭）”位置。
    • 6.3 测试控制电路端对端的电阻是否小于2 Ω。
    • 如果为 2 Ω 或更大，则修理电路中的开路/电阻过大故障。
    • 如果小于2 Ω。
  • 如果故障指示灯点亮
  • 7.将点火开关置于“OFF（关闭）”位置，断开组合仪表的线束连接器。
  • 8.将点火开关置于“ON（打开）”位置，确认点火电压电路端子5和搭铁之间的测试灯点亮。
  • 如果测试灯未点亮且电路保险丝熔断
    • 8.1 将点火开关置于“OFF（关闭）”位置。
    • 8.2 测试点火电压电路和搭铁之间的电阻是否为无穷大。
    • 如果电阻不为无穷大，则修理电路中的对搭铁短路故障。
  • 如果测试灯未点亮且电路保险丝完好
    • 8.1 将点火开关置于“OFF（关闭）”位置。
    • 8.2 测试点火电压电路端到端的电阻是否小于2 Ω。
    • 如果为 2 Ω 或更大，则修理电路中的开路/电阻过大故障。
    • 如果小于2 Ω ，则更换K20 发动机控制模块
  • 如果测试灯点亮
  • 9.更换 P16 组合仪表。
