无钥匙进入系统的说明与操作

无钥匙进入系统的说明与操作－主动式

无钥匙进入系统是一个车辆进入装置。同时使用无钥匙进入系统和门锁解锁车辆。当按下无钥匙进入发射器上相应的按钮时，无钥匙进入系统将锁止/解锁车门或打开行李厢盖。这是通过由发射器向遥控门锁接收器天线发射无线电频率实现的，遥控门锁接收器天线与车身控制模块（BCM）直接链路。车身控制模块解读该信号并启动请求的功能或请求相应的电子控制单元通过串行数据信息启动功能。发射器蓄电池电量过低或来自售后加装设备（如双声道收音机、电源逆变器、计算机、12 V电源插座USB适配器等）的无线电频率的干扰，可导致系统故障。在无线电频率通信量较大的区域，例如使用无线电频率收发器刷卡付费的加油站，产生的干扰也可导致故障。无钥匙进入系统允许操作以下功能：
  • •  车门锁止/解锁
  • •  行李厢盖释放
  • •  上车照明灯
  • •  防盗警报/车辆定位器
  • •  车辆遥控启动
  • •  自动车窗快速下降（如装备）。自动车窗快速上升（如装备）

无钥匙进入系统的说明与操作－无钥匙起动

无钥匙进入系统的无钥匙起动部分仅将发射器作为钥匙，允许车辆起动。无钥匙起动系统使用位于车辆上3个不同位置的低频天线来确定发射器的位置。使用了多个天线，确保完全覆盖车辆内部和行李厢。使用无钥匙起动系统时，车内、驾驶员衣兜、钱包或公文包内必须有一个已编程的发射器。
按下点火模式开关后，低频天线将向无钥匙进入发射器发送一个校验口令。发射器接收该校验口令，然后发射无线电频率响应信息给遥控门锁接收器。如果响应正确，则将允许车辆起动。如果无线电频率 (RF) 通信中断，则驾驶员信息中心将显示信息“No Remote Detected（未检测到遥控钥匙）”。这些情况下，可将发射器置于中控台的发射器槽内，杯架下方。安全防盗系统天线线圈位于发射器凹槽正下方。将发射器置于凹槽中会在发射器和安全防盗系统天线之间生成一个低功率的耦合，允许出现通信并使车辆起动。
无钥匙进入系统有以下部件：
  • •  无钥匙进入系统集成的钥匙/发射器
  • •  前控制台天线
  • •  车身控制模块 (BCM)
  • •  安全防盗系统天线线圈总成
  • •  遥控门锁接收器
车门天线
无钥匙进入车门天线用于将低频通信传输至无钥匙进入发射器。
无钥匙进入车门天线位于左前和右前车门外把手托架中。天线作为车门把手总成的一部分进行维修。天线由无钥匙进入控制模块控制。当按下车门外把手按钮时，相应的天线向无钥匙进入发射器发送校验口令，开始被动式进入车辆的通信。
前控制台被动（仪表板）天线
该天线位于中控台前方附近的仪表板下部。
前控制台天线用于车辆起动功能。按下点火模式开关时，天线将通电或发送信号。这样会发送一个低频校验口令信号给无钥匙进入发射器。发射器随后将响应该校验口令，如果正确，则车辆将起动。
车身控制模块 (BCM)
车身控制模块是一个多功能模块，可执行以下功能：
  • •  接收并验证遥控门锁接收器的激活发射器和无钥匙起动信号
  • •  确定发射器信号请求的功能
  • •  执行发射器信号请求的功能
安全防盗系统天线线圈总成
安全防盗系统线圈总成位于中控台中，发射器凹槽的正下方。如果发射器蓄电池没电、电流弱或无线电频率信号受到干扰，则可将发射器置于凹槽内，以在发射器和安全防盗系统天线之间生成一个低功率耦合，允许出现通信并使车辆起动。
安全防盗系统天线线圈受车身控制模块控制。读入车辆钥匙时安全防盗系统控制模块也将激活。
安吉星® 远程链接（如装备）
车辆操作人员可以使用如（但不限于）智能电话等个人设备上的应用软件执行一些无钥匙进入功能。可能出现车辆似乎存在故障的情况。如果怀疑情况可能如此并且无法通过正常的车辆诊断确定故障，请联系技术援助人员。
仅解锁驾驶员车门－主动式
快速按下发射器“UNLOCK（解锁）”按钮，以执行以下功能：
  • •  仅解锁驾驶员车门
  • •  点亮车内灯至指定的时间，或直至点火开关置于“ON（打开）”位置
  • •  如果通过个性化启用，则车外灯闪烁
  • •  解除安全防盗（CTD）系统
  • •  在警报模式时，停用安全防盗系统
解锁所有车门－第二次操作－主动式
再次快速按下发射器“UNLOCK（解锁）”按钮（在第一次按下后5s内），以执行以下功能：
解锁其余的车门
锁闭所有车门－主动式
按下发射器的“LOCK（锁止）”按钮，以执行以下功能：
  • •  锁止所有车门
  • •  立即熄灭车内灯
  • •  如果通过个性化启用，则车外灯闪烁
  • •  如果通过个性化启用，则喇叭发出唧唧声
  • •  启动安全防盗系统
行李厢盖释放－主动式
如果车辆变速驱动桥挂驻车档或空档，且将点火开关置于“OFF（关闭）”位置，则按一下发射器行李厢释放按钮将打开行李厢盖。
车辆定位器/防盗警报/主动式
按一下防盗按钮可执行以下功能。其中某些功能可能取决于个性化设置：
  • •  使喇叭鸣响3次
  • •  使车外灯闪烁3次
按住防盗按钮可执行以下功能：
  • •  按住防盗按钮可执行以下功能：
  • •  喇叭鸣响且驻车灯闪烁30s，或直到以下情况发生：
    • – 按下防盗按钮
    • – 使用一个有效的钥匙将点火开关置于RUN（运行）位置
车辆遥控启动/启用
车辆遥控启动功能允许不在车内时起动发动机。它同时允许启用车辆“暖风、通风与空调”系统和其他车辆系统，为上车提供舒适的车内环境。通过按下并松开锁止按钮，然后按住无钥匙进入发射器上的车辆遥控启动按钮，可完成车辆遥控启动。转向信号灯会点亮，以指示车辆已经接收到遥控启动请求。每次进行车辆遥控启动，车门都会锁止，然而之后随时可用发射器或车辆钥匙解锁/锁止。仅首个和第二个车辆发射器可控制车辆遥控启动功能。任何编程至车辆的附加遥控发射器将执行所有其他遥控功能。一旦启用，即让发动机运转10min。通过再次按下和释放锁止按钮，然后按住发射器上的车辆遥控启动按钮，车辆遥控启动时间可再延长10min。此功能称为车辆遥控启动延续，它允许发动机运转最长20min。如果在最初的10min倒计时内的第7min进行车辆遥控启动延续，将重新进行10min倒计时，发动机一共可运转17min。仅按下发射器上的车辆遥控启动按钮或进入车辆并打开危险警告时，遥控车辆起动可能被中止。
在点火循环之间，只能发生或实现2次车辆遥控启动。一旦已经发生或尝试2次，以后的车辆遥控启动都将中止，直到用点火开关起动车辆。
启用/停用车辆遥控启动/激活
使用驾驶员信息中心车辆遥控启动可作为车辆个性化设置的一部分启用或停用。更多信息请参见车主手册。
发动机舱盖微开开关/激活
发动机舱盖开关向车身控制模块提供发动机舱盖的状态，以用于车辆遥控启动。该开关集成在发动机舱盖锁闩总成内。发动机舱盖微开开关向车身控制模块提供2个独立的输入。这些独立的输入使车身控制模块能主动监测发动机舱盖微开开关故障。
车辆遥控启动/激活电路说明
车身控制系统接收到来自无钥匙进入发射器的信号，指示车辆遥控启动请求。车身控制模块使用以下输入以确认系统是否为车辆遥控启动工作做好准备：
  • •  车辆不处于车童模式
  • •  所有车门都已关闭
  • •  行李厢盖已关严
  • •  发动机舱盖已关闭
  • •  车门已锁止
  • •  低油位警告灯熄灭
  • •  危险警告开关处于“OFF（关闭）”位置
  • •  未重新连接蓄电池
  • •  车辆电源模式正确
当车身控制模块 (BCM) 确定车辆遥控启动所需的所有条件都得到满足时，通过串行数据发送消息至发动机控制模块 (ECM)。收到起动请求信号时，发动机控制模块 (ECM) 依据来自车身控制模块 (BCM) 的车辆遥控启动消息启用车辆遥控启动。如果发动机控制模块未接收到有效的车辆遥控启动消息，将不能使起动继电器的控制电路搭铁并起动发动机。发动机控制模块处于车辆遥控启动模式时，如果发生了任何下列其他情况，将切断发动机燃料：
  • •  车速大于0
  • •  变速器不处于驻车挡
  • •  发动机冷却液温度过高
  • •  机油压力过低
  • •  指令故障指示灯（MIL）点亮
  • •  发动机起动时间大于30s
  • •  发动机转速过大
  • •  加速踏板位置过高
  • •  遥控启动计时器等于0
  • •  安全防盗系统指示盗车企图
