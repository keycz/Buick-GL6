电路说明
故障指示灯(MIL) 也称为立即维修发动机(SES) 或检查发动机指示灯，位于组合仪表上。点亮时，故障指示灯通知驾驶员出现排放系统故障，需要对系统进行维修。 点火电路块向指示灯提供电源，发动机控制模块(ECM)提供开关搭铁。

电路/ 系统检验
注意:应先诊断故障指示灯请求可能设置的任何故障诊断码。
  • 1.将点火开关置于“ON （打开）”位置。
  • 2.用故障诊断仪指令故障指示灯点亮和熄灭。
  • 3.确认故障指示灯按照指令点亮、熄灭。
  • 如果故障指示灯未按照指令点亮、熄灭
    参见 电路测试
  • 如果故障指示灯按照指令点亮、熄灭
  • 4.将点火开关置于“ON （打开）”位置。
  • 5.用故障诊断仪指令故障指示灯点亮和熄灭，并确认下列控制电路状态参数显示“OK （正常）” 或“Not Run （未运行）”。
    • •  MIL Control Circuit Low Voltage Test Status （故障指示灯控制电路电压过低测试状态）
    • •  MIL Control Circuit Open Test Status（故障指示灯控制电路开路测试状态）
    • •  MIL Control Circuit High Voltage Test Status （故障指示灯控制电路电压过高测试状态）
  • 如果显示故障
    参见 电路测试
  • 如果显示“OK （正常）”或“Not Run （未运行）”
  • 6.发动机运转。
  • 7.用故障诊断仪指令故障指示灯点亮和熄灭，并确认下列控制电路状态参数显示“OK （正常）” 或“Not Run （未运行）”。
    • •  MIL Control Circuit Low Voltage Test Status （故障指示灯控制电路电压过低测试状态）
    • •  MIL Control Circuit Open Test Status（故障指示灯控制电路开路测试状态）
    • •  MIL Control Circuit High Voltage Test Status （故障指示灯控制电路电压过高测试状态）
  • 如果显示故障
    参见 电路测试
  • 如果显示“OK （正常）”或“Not Run （未运行）”
  • 8.全部正常。
电路/ 系统测试
  • 1.将点火开关置于“OFF （关闭）”位置。
  • 2.断开K20 发动机控制模块处的X1 线束连接器。
  • 3.将点火开关置于“ON （打开）”位置。
  • 4.确认故障指示灯未点亮。
  • 如果故障指示灯点亮
    • 4.1 测试故障指示灯控制电路端子X1 D4和搭铁之间的电阻是否为无穷大。
    • 如果电阻不为无穷大，则修理电路上的对搭铁短路故障。
    • 如果电阻无穷大，更换P16 组合仪表。
  • 如果故障指示灯未点亮
  • 5.在故障指示灯控制电路端子X1 D4和搭铁之间连接一根带3 安培保险丝的跨接线。
  • 6.确认故障指示灯点亮。
  • 如果故障指示灯未点亮
    • 6.1 测试控制电路端子 X1 D4和搭铁之间的电压是否小于1 V。
    • 如果等于或大于1 V，则修理电路中的对电压短路故障。
    • 如果小于1 V
    • 6.2 将点火开关置于“OFF（关闭）”位置。
    • 6.3 测试控制电路端对端的电阻是否小于2 Ω。
    • 如果为 2 Ω 或更大，则修理电路中的开路/电阻过大故障。
    • 如果小于2 Ω。
  • 如果故障指示灯点亮
  • 7.将点火开关置于“OFF（关闭）”位置，断开组合仪表的线束连接器。
  • 8.将点火开关置于“ON（打开）”位置，确认点火电压电路端子5和搭铁之间的测试灯点亮。
  • 如果测试灯未点亮且电路保险丝熔断
    • 8.1 将点火开关置于“OFF（关闭）”位置。
    • 8.2 测试点火电压电路和搭铁之间的电阻是否为无穷大。
    • 如果电阻不为无穷大，则修理电路中的对搭铁短路故障。
  • 如果测试灯未点亮且电路保险丝完好
    • 8.1 将点火开关置于“OFF（关闭）”位置。
    • 8.2 测试点火电压电路端到端的电阻是否小于2 Ω。
    • 如果为 2 Ω 或更大，则修理电路中的开路/电阻过大故障。
    • 如果小于2 Ω ，则更换K20 发动机控制模块
  • 如果测试灯点亮
  • 9.更换 P16 组合仪表。
